# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Open a private security advisory in GitHub if available, or contact the repository owner through the GitHub profile. Include reproduction steps, affected files, expected impact, and any relevant logs.

Do not include secrets, credentials, or private system data in reports.
